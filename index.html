Tilt/orientation sensor system

Developed a 2D orientation sensing system using an ADXL335 analog sensor and an Arduino with a successive approximation ADC PCB.

  1. Created a prototype on a Vero-board and tested it with an Arduino.
  2. Designed the ADC on Multisim and had the PCB manufactured.
  3. Acquired and processed real-time analog voltage signals from the accelerometer's X and Y axes to determine board tilt.
  4. Implemented mathematical formulas using an Arduino to convert voltage outputs into angle measurements representing the vertical and horizontal orientation of the PCB.

Weather station

Designed and implemented a smart weather monitoring system using multiple communication protocols for real-time data acquisition and transmission.

  1. The BME280 and DS18B20 sensors measure temperature, pressure, and humidity with the readings displayed on an LCD via I2C.
  2. SPI was used for LoRa integration which enables long-range transmission of the readings from the sensors.
  3. Wi-Fi hosts a web server to display sensor data in real time.
  4. Bluetooth allows readings to be accessed on a mobile device through a serial terminal app.
